Methodology
Projections are built from Vegas prop market alt ladders via Monte Carlo simulation, blended with an internal model using snap share, target share, and role data. Start/Sit accuracy is scored binary: correct if the player started outperformed the alternative. MAE is mean absolute error in PPR points across all numeric projections.
